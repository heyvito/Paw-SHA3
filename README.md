# SHA3 Digest (Paw Extension)
This is a very simple [Paw Extension](http://luckymarmot.com/paw/extensions/) to hash data using SHA3
algorithms. Supported algorithms are:
 - SHA3-512
 - SHA3-384
 - SHA3-256
 - SHA3-224
 - Keccak-512
 - Keccak-384
 - Keccak-256
 - Keccak-224
 - Shake-128
 - Shake-256
